Налаштування php.ini:

                    ВИВЕДЕННЯ ПОМИЛОК В PHP
Все настройки вывода ошибок находятся в разделе Error handling and logging. 
	По умолчанию вывод ошибок на экран во время выполнения скрипта отключен (все ошибки записываются в лог файл) - display_errors=off

1 - Щоб включити виведення помилок на екран - міняємо на off on - display_errors=on

2 - Виведення лише критичних помилок - error_reporting = E_ALL  & ~E_DEPRECATED - 

3 - Відключити всі помилки - error_reporting = 0

4 - Включите запись ошибок php в лог файл, если не выводите их на экран - 
      log_errors = On

5 - игнорировать повторяющиеся ошибки в пределах одного исполнения - 
		ignore_repeated_errors = On

		            ОГРАНИЧЕНИЯ РЕСУРСОВ
6 - максимальное время выполнения скрипта (По умолчанию - 30 сек),  робимо 			1 хв. - max_execution_time = 60 (якщо 0, то викон. може бути вічно)

7 - Макс. к-ть змінних в GET и POST  -  max_input_vars = 1000

8 - Макс. к-ть  пам'яти, которую может использовать один скрипт во время своего выполнения, в мегабайтах  - memory_limit = 128M

9 - Макс. размір даних, перед. в POST запросі обмежено (за замовч. - 8 Мб)
	post_max_size = 8M

10 - обмежити область дії php в системі  - папка, вище якої скрипт не может получити доступ до файл. сист.  -  open_basedir = /var/www/

11 - С помощью директив disable_functions и disable_classes вы можете отключить использование в скриптах определенных функций или классов, например, это может быть полезно для веб-хостингов. 
	В этом примере мы отключаем использование функции ini_set, которая позволяет менять настройки php из скрипта - disable_functions = ini_set

12 - задать папки где система будет искать скрипты, если вы попытаетесь подключить их с помощью инструкции include - 
include_path = ".:/usr/share/php5:/usr/share/php5/PEAR"

13 - Папка с модулями php - extension_dir="./"

14 - Папка для записи временных файлов - sys_temp_dir = "/tmp"

						ЗАГРУЗКА ФАЙЛОВ
15 - Для того чтобы пользователи могли загружать свои файлы на сервер, например, фото, нужно включить эту функцию в php - file_uploads = On

16 - Максимальный размер загружаемого файла - upload_max_filesize = 2M

17 - Максимальное количество файлов, которые может загрузить один скрипт - 
		max_file_uploads = 20

				НАСТРОЙКА РАСШИРЕНИЙ
18 - Расширения позволяют очень сильно увеличить функциональность php. Например, благодаря расширениям вы можете использовать в своих скриптах базы данных mysql, postgresql, mysqli, sqlite, графическую библиотеку gd и многое другое. Все это включается в этом разделе.

Для включения расширения достаточно убрать комментарий перед строкой с его командой, например:
  -   extension=php_mysql.so
  -   extension=php_mbstring.so
  -   extension=php_pgsql.so
Обратите внимание, что для windows расширение будет иметь формат dll, но для linux нужно использовать so
